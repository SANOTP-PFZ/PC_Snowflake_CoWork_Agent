## Context

**Current app**: Streamlit-based (`DSS_app.py`) with custom CSS, flat light-blue palette, simple card layouts, no sidebar, header + screen routing.

**Target (Migraine Hub)**: Pure HTML app with rich design — navy color palette, Manrope+Inter fonts, glassmorphism (backdrop-filter blur), gradient icon chips, elevated card hover effects with gradient overlays, refined shadow system, and polished spacing.

**Key constraint**: Streamlit can't replicate a sidebar navigation or animated section transitions. But we CAN adopt the visual language — colors, fonts, card styling, shadows, backgrounds, and overall polish.

## Implementation Steps

### Step 1: Update CSS Variables (color palette + shadows)
In `DSS_app.py`, replace the `:root` block with the Migraine Hub's navy-based palette:
- `--bg: #EEF3FB` (lighter, more refined)
- `--surface: #FFFFFF`
- `--text-1: #0F172A` (navy-dark)
- `--text-2: #1C4FC0` (navy-600)
- `--text-3: #64748B` (muted)
- `--accent: #41B6E6` (cyan-blue)
- `--accent-soft: rgba(65,182,230,0.08)`
- New shadow scale: `--shadow-sm`, `--shadow-md`, `--shadow-lg` matching the Hub's refined values
- `--radius: 14px`, `--radius-lg: 18px`

### Step 2: Add Google Fonts (Manrope + Inter)
Inject a `<link>` tag for Manrope and Inter fonts. Apply Manrope to headings (`h1-h4, .hub-title, .ta-card-title`) and Inter to body text.

### Step 3: Update background to use subtle radial gradients
Replace flat `--bg` color with layered radial gradients matching the Hub:
```css
background:
  radial-gradient(ellipse 80% 60% at 0% 0%, rgba(28,79,192,0.06) 0%, transparent 60%),
  radial-gradient(ellipse 70% 50% at 100% 0%, rgba(65,182,230,0.05) 0%, transparent 55%),
  var(--bg);
```

### Step 4: Update header to glassmorphism style
Change `.hub-header` to use `backdrop-filter: saturate(180%) blur(22px)` with semi-transparent white background (`rgba(255,255,255,0.62)`), matching the Hub sidebar panel style.

### Step 5: Restyle agent cards to match Hub card design
- Add gradient overlay on hover (`::after` pseudo-element)
- Use refined shadow elevation (`shadow-sm` → `shadow-lg` on hover)
- Add `translateY(-3px)` hover with smoother cubic-bezier easing
- Update card title to use Manrope font, 700 weight
- Update card description color to `--text-3`

### Step 6: Restyle category cards on landing page
- Match the Hub's card style with gradient icon chip backgrounds
- Use the `card-top` pattern with colored icon chips
- Add subtle gradient overlay on hover

### Step 7: Update brand trend boxes
- Add glassmorphism: `background: rgba(255,255,255,0.75); backdrop-filter: blur(8px)`
- Refine border and shadow to match Hub KPI cards

### Step 8: Update buttons and pills
- Refine button styling to match the Hub's `icon-btn` pattern (lighter, cleaner borders)
- Active pill: use navy-700 background instead of bright accent

### Step 9: Update footer
- Match the Hub's refined, minimal footer style with backdrop-filter

### Step 10: Polish transitions
- Replace `* { transition: all 0.15s ease; }` with targeted transitions using `cubic-bezier(0.4, 0, 0.2, 1)` on specific elements (cards, buttons) to avoid jank on layout elements.
